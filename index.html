The dashboard could not fetch one or more Excel files. This usually means the GitHub URLs in config.js need to be updated with your real file URLs.

Make sure your GitHub repository is public and you're using the raw file URL format:

https://raw.githubusercontent.com/YOUR-USERNAME/YOUR-REPO/main/EC_Audit_Andheri.xlsx

See the setup guide below for full instructions.

Setup guide — 10 minutes

Follow these steps once and your dashboard will auto-update whenever you upload new Excel files to GitHub.

1

Create a free GitHub account

Go to github.com → Sign up → choose the free plan.

2

Create a new repository

Click the + icon → New repository → name it ec-audit → set to Public → click Create.

3

Upload your 4 Excel files + this HTML file

In your repo, click Add file → Upload files → drag all 5 files:
EC_Audit_Andheri.xlsx   EC_Audit_Thane.xlsx   EC_Audit_HSR.xlsx   EC_Audit_HRBR.xlsx   index.html
Click Commit changes.

4

Enable GitHub Pages

Go to repo Settings → Pages → Source → Deploy from branch → main → / (root) → Save.
Your dashboard URL will be: https://YOUR-USERNAME.github.io/ec-audit/

5

Update the file URLs inside index.html

Open index.html in any text editor. Find the FILE_URLS section near the top of the script and replace YOUR-USERNAME and YOUR-REPO with your actual GitHub username and repo name.

const FILE_URLS = { Andheri: 'https://raw.githubusercontent.com/YOUR-USERNAME/ec-audit/main/EC_Audit_Andheri.xlsx', Thane: 'https://raw.githubusercontent.com/YOUR-USERNAME/ec-audit/main/EC_Audit_Thane.xlsx', HSR: 'https://raw.githubusercontent.com/YOUR-USERNAME/ec-audit/main/EC_Audit_HSR.xlsx', HRBR: 'https://raw.githubusercontent.com/YOUR-USERNAME/ec-audit/main/EC_Audit_HRBR.xlsx', };

Save the file and re-upload it to GitHub.

6

To update the dashboard in future

Update the Excel file on your computer → go to GitHub → click the file → click the pencil/upload icon → upload the new version → Commit. The dashboard URL stays the same and shows new data immediately.
